## Overview

### Platforms

| Platform          | Requirements              | Review Time | Update Frequency      |
| ----------------- | ------------------------- | ----------- | --------------------- |
| iOS App Store     | Strict review process     | 24-48 hours | Bi-weekly recommended |
| Google Play Store | Automated + manual review | 2-7 days    | Weekly recommended    |

### Quality Standards

- **Screenshots:** High-resolution, actual app content (no mockups)
- **Videos:** 30 seconds max, show core features
- **Icons:** Follow platform guidelines strictly
- **Descriptions:** Clear, concise, keyword-optimized
- **Metadata:** Accurate, complete, localized

## App Preview Videos

### iOS App Preview

**Requirements:**

- Length: 15-30 seconds
- Format: .mov, .m4v, or .mp4
- Resolution: Match screenshot sizes
- File size: Max 500MB
- No audio required (but recommended)

**Recommended Sizes:**

- iPhone 6.7": 886 Ã— 1920 or 1080 Ã— 1920
- iPhone 6.5": 886 Ã— 1920 or 1080 Ã— 1920
- iPad Pro: 1200 Ã— 1600

### Android Promo Video

**Requirements:**

- Length: 30 seconds to 2 minutes
- Format: YouTube video URL
- Aspect ratio: 16:9
- Resolution: 1920 x 1080 (Full HD)

### Video Content Script

**Duration: 30 seconds**

```
[0-3s] Opening: App logo animation with tagline
"UpCoach - Your Personal Growth Companion"

[3-8s] Problem: Quick visual of common struggles
Show: Cluttered to-do lists, forgotten goals, lack of motivation

[8-15s] Solution: App features montage
- Quick goal creation (2s)
- Habit tracking with streak (2s)
- AI insight notification (2s)
- Progress chart animation (2s)

[15-25s] Benefits: User testimonial or stats
"Join 10,000+ users achieving their goals"
Show: Happy user checking off habit, celebrating milestone

[25-30s] Call to Action
"Download UpCoach Today - Free"
App icon + Download buttons
```

### Video Production Tools

**Screen Recording:**

- iOS: Built-in screen recording
- Android: AZ Screen Recorder
- Desktop: OBS Studio, QuickTime

**Editing:**

- Free: iMovie, DaVinci Resolve
- Paid: Adobe Premiere, Final Cut Pro
- Online: Kapwing, Clipchamp

**Tips:**

1. Use actual app footage (no simulations)
2. Show finger taps/interactions
3. Add subtle background music
4. Include captions for accessibility
5. End with clear call-to-action
6. Test on device before uploading

### iOS Icon Sizes

**Required:**

| Size      | Usage     | Pixels      |
| --------- | --------- | ----------- |
| 1024x1024 | App Store | 1024 x 1024 |
| 180x180   | iPhone 3x | 180 x 180   |
| 120x120   | iPhone 2x | 120 x 120   |
| 167x167   | iPad Pro  | 167 x 167   |
| 152x152   | iPad 2x   | 152 x 152   |
| 76x76     | iPad 1x   | 76 x 76     |

### Android Icon Sizes

**Required:**

| Density | Size | Pixels    |
| ------- | ---- | --------- |
| xxxhdpi | 4x   | 192 x 192 |
| xxhdpi  | 3x   | 144 x 144 |
| xhdpi   | 2x   | 96 x 96   |
| hdpi    | 1.5x | 72 x 72   |
| mdpi    | 1x   | 48 x 48   |

**Google Play:**

- High-res icon: 512 x 512 (PNG, 32-bit)

### Icon Design Guidelines

**iOS:**

- No transparency
- Square with rounded corners (system applies)
- 1024x1024 for submission
- Simple, recognizable at small sizes

**Android:**

- Adaptive icon: Foreground + Background layers
- Foreground: 108 x 108 dp (within 66dp safe zone)
- Background: 108 x 108 dp
- Legacy icon: 512 x 512 (PNG)

### Icon Design Checklist

- [ ] Recognizable at 40x40 pixels
- [ ] Works on light and dark backgrounds
- [ ] No text (unless essential to brand)
- [ ] Consistent with brand colors
- [ ] Simple, not cluttered
- [ ] Tested on home screen
- [ ] Follows platform guidelines
- [ ] Exported in all required sizes

### Icon Generator Tools

- [https://appicon.co/](https://appicon.co/) - Generate all sizes
- [https://icon.kitchen/](https://icon.kitchen/) - Adaptive icon generator
- [https://www.figma.com/](https://www.figma.com/) - Design in Figma
- Flutter: `flutter_launcher_icons` package

```yaml

flutter pub run flutter_launcher_icons
```

### App Store Optimization (ASO)

**Keyword Research:**

```
Primary Keywords:
- habit tracker
- goal setting
- life coach
- personal development
- productivity app

Secondary Keywords:
- habit building
- goal tracker
- self improvement
- daily planner
- motivation app

Long-tail Keywords:
- habit tracker with reminders
- goal setting and tracking
- personal growth journal
- life coaching app free
```

**Tools:**

- App Annie
- Sensor Tower
- Mobile Action
- AppTweak

**Optimization Tips:**

1. **Title Optimization:**
   - Include primary keyword
   - Stay within character limits
   - Make it memorable

2. **Description Optimization:**
   - Front-load important keywords
   - Use bullet points for features
   - Include social proof (ratings/reviews)
   - Add clear call-to-action

3. **Visual Optimization:**
   - First screenshot is most important
   - Show value immediately
   - Use text overlays sparingly
   - A/B test different versions

4. **Rating Optimization:**
   - Prompt for reviews at right moment
   - Respond to all reviews
   - Fix issues mentioned in reviews
   - Maintain 4.5+ rating

## Localization

### Priority Languages

**Tier 1 (High Priority):**

1. English (US) - Primary
2. Spanish (ES, MX)
3. French (FR)
4. German (DE)
5. Portuguese (BR)

**Tier 2 (Medium Priority):** 6. Japanese (JP) 7. Korean (KR) 8. Italian (IT) 9. Dutch (NL) 10.
Russian (RU)

### Localization Requirements

**For Each Language:**

1. **App Store Metadata:**
   - App name (translated if culturally appropriate)
   - Description (fully translated)
   - Keywords (researched for that market)
   - Screenshots (text overlays translated)

2. **In-App Content:**
   - All UI text
   - Error messages
   - Onboarding flow
   - Push notifications
   - Email templates

3. **Marketing:**
   - Website landing pages
   - Social media posts
   - Support documentation
   - FAQs

### Localization Tools

- **Translation:**
  - Professional: Lokalise, Phrase
  - Community: Crowdin
  - Machine: Google Translate (review required)

- **Testing:**
  - Native speakers
  - In-market beta testers
  - Localization QA services

### Cultural Considerations

**Colors:**

- Red: Lucky in China, danger in Western cultures
- White: Purity in West, mourning in East Asia

**Icons:**

- Thumbs up: Offensive in some Middle Eastern countries
- Hand gestures: Research before using

**Text:**

- Right-to-left languages (Arabic, Hebrew): UI must mirror
- Text expansion: German ~30% longer than English
- Date formats: MM/DD/YYYY vs DD/MM/YYYY

## Next Steps

1. **Design Phase (Week 1-2):**
   - [ ] Create app icon designs (3 variations)
   - [ ] Design screenshot templates
   - [ ] Write all copy/descriptions
   - [ ] Get stakeholder approval

2. **Production Phase (Week 3-4):**
   - [ ] Generate all icon sizes

   - [ ] Add text overlays to screenshots

   - [ ] Translate metadata (Tier 1 languages)
